MTG Draft Simulator


  • Built a program that allows one to simulate a Magic: The Gathering booster draft
  • Uploaded data from an excel spreadsheet, consisting of data such as the name and the ratings of the cards to a SQLite database
  • The program will randomly select cards from SQLite database, insert them into lists, and will select and remove cards from each list based on how high the rating is using if/else statements and while/for loops. The user in the meantime will also select cards to add to their existing deck
